# AutoFounder 🤖

> The fully autonomous AI business builder. Build, run, and scale your business 24/7.

## What it does
- 🧠 Takes your business idea and generates a full strategy
- 💻 Builds and deploys a real web app to Vercel automatically  
- 🚀 Runs autonomous operations 24/7 — tasks, content, lead gen
- 📊 Tracks metrics and sends daily performance reports

## Stack
- **Frontend:** React + Vite + Tailwind CSS
- **Backend:** Base44 (Deno functions)
- **AI:** Nvidia NIM (Llama 3.3 70B)
- **Deploy:** GitHub + Vercel

## Getting Started
```bash
npm install
npm run dev
```
